# 关于
本节介绍C++的模板编程，引入IntegerFormatter类用于将格式化数字。

C++模板编程包括两个部分：类模板和函数模板。

由于模板不是真正的代码，因为含有待定的参数，所以模板不参与编译。

只有模板实例化时，模板的代码才会被编译，因此需要模板代码在引入时可见。
